Instructions

  1. Open Hootsuite, login with aqreporting@gov.bc.ca (Password is the same as AQ Reporting IDIR password.)

  2. Click on the down arrow of the New Post button, and select "Open Legacy Composer"

  3. Insert the URL link in the URL field and click on "Shrink".

  4. Copy the newly created shortened URL link and return to the AQSS posting page to paste the link along with your SMS text message.
